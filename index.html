All Files/Plugins

LX's Plugins have been created while learning Bukkit and Java coding These plugins contain code which has been developed and tested to help understand Bukkit and Java Code. These plugins may be used on a Minecraft Server With Permission From the Creator LX_Gaming (Alex Thomson)

Simply Type Commands In-Game Check the link below for a list of commands
